访谈记录-关于烹饪

发表于 2017-02-06 | 分类于 调研

背景

越来越多的家电引入人工智能和大数据来提升产品体验，本访谈的初衷是为国内某家电企业的智能冰箱做关于烹饪场景的用户调研。

截取部分访谈内容和总结。

访谈目标

了解用户做菜的习惯和使用菜谱的情况

过程

面对面预先沟通，询问对方是否有访谈的意向，微信预定初步访谈时间，微信确定访谈时间及访谈计划

类型

面对面半结构式访谈

访谈大纲

  • 1.描述做饭的过程
  • 2.做饭的初衷
  • 3.对食材的要求
  • 4.做饭的学习过程
  • 5.买菜的习惯

开场白

首先非常感谢您来参加这次用户访谈
你是第一次做这样的访谈的访谈吧，别紧张我们只是聊聊做饭这件事，代表广大小白请教请教，不会涉及任何隐私，如果一些事情你不方便回答，也可以不回答
过程中可能会有录音，但请您放心，我们不会涉及任何个人隐私，今天只是想跟你请教如何做饭这件事

问题大纲

  • 您经常自己做饭吗
  • 描述一下你做饭的经过
  • 因为什么原因自己做饭
  • 从什么开始做饭的
  • 如何学会做饭的
  • 如何学会一道新菜式
  • 学习菜谱您一般在哪些平台 （下厨房 豆果用过）
  • 您觉得做饭最大的困难吗，困难/简单。为什么这么说
  • 需要哪些注意哪些要点（我经常把锅烧黑）
  • 教程中的菜谱，你都会顺利完成吗？（火候什么的比较难把握）
  • 对做饭的食材有什么要求

结束语

感谢您百忙之中抽出时间来做用户调研，今天的访谈非常愉快，时间不早了，就不耽误你时间了，您方便留下您的联系方式吗，邮箱也行，这样我可以版调研报告发给你，也许会做一个跟踪调研

访谈对象

对象 年龄段 工作
室友Luo 20-25 外企白领

访谈内容

  • Q:你是每天都很早准时下班回来做饭？
  • A:因为我喜欢做饭，下班了还待在办公室里，作息和用餐还蛮准时的，到点了就想吃饭，然后如果在办公室加班的话呢就叫外卖，但我不想叫外卖，但同时不是那种必须我加班的事，我一般都会回来，把工作带回来。
  • Q:工作也可以回来做是吧？
  • A:是的。其实不是不加班而是有加班，但是我会回来的话，我会先回来吃饭。
  • Q:你还是挺喜欢做饭的哈
  • A:恩！因为觉得自己吃是一方面，第二个方面就是你做饭给别人吃，比如给身边的人吃，他吃完之后觉得很好吃，然后你就会特别有成就感。
  • Q:那你一般性做饭的话，你是怎么做的？能描述一下？
  • A:其实我上班了，刚好我们单位有一批女孩子，她们喜欢做饭，然后就和她们交流，中午她们全带饭，后来我们就自己做饭。
  • Q:受周围人的影响
  •  A:是有点影响，然后就开始做饭
  • Q:我们公司有些同事她也做饭，就是比较少 
  • A:因为我们不是商业公司，然后团队里面氛围就很好，因为我们这个单位他其实以项目团队组成的，所以就是自己小团队的话就是十几二十个人，但是其实还有更大的团队，更大的团队有时候就会聚餐，不对，小团队再具体到项目的话就三五个人，但是比如你那个部门叫大客户，大客户加起来可能有二十多人，所以我们每个月会举行一次冷餐会，其实冷餐会并不是说吃冷的，（大家自己做的饭）必须是自己做的饭，不许带买卖，不许叫外卖，所以不管难吃不难吃，然后我们就都做饭
  • Q:男同事也做吗
  • A:男同事会啊，因为我们这边男同事不是特别多，但是有两个刚好是上海本地的，然后他就是做蛋糕啊酸奶啊烤什么东西特别好，所以这个活动也是他发起的
  • Q:我们公司有个女同事也很会做饭，她做的起司蛋糕就很好吃，还有自己做的牛轧糖
  • A:所以同事之间是会相互影响的
  • Q:所以那你们公司有食堂吗
  • A:有，我们现在换办公室了，之前也没有，之前园区里面有个大食堂，现在换办公室了呢，是一个社创空间，你有听过吗；社创空间它下面活动室和办公室，上面是零时的办公位一样的，它就是给那些创业公司的。人家是商业创业，我们是公益创业社会创业，也是我们单位弄了个空间，下面是小剧场图书馆，上面就会有咖啡厅，快餐，然后关键是有一个功能很好的茶水间，有烤箱之类的我们平时也可以在那边做饭，所以有时候，会包包饺子啊烤烤蛋挞，关键是我们那边有一个非常贤惠的小姑娘每次都给我们做好吃的，象征性每个人出一点材料费就可以了
  • Q:买菜的话是每天都买吗
  • A:不是，买菜的话，现在不太方便了，刚搬过来，住到这边之前，小区里有一个app叫食航生鲜，它上面可以卖东西的，然后你可以今天订，明天他可以送到小区门口的冷藏柜，你凭密码就可以取菜，因为针对我们下班很晚的人真的不要太方便，要不然去菜场，第一菜不新鲜第二要不已经关门了
  • Q:我们小区也有吗
  • A:我们小区没有，所以现在很不方便，我已经用这个app一年多了，现在已经习惯了在那边买菜，所以我会查了附近的点，就是地铁站那边的，所以有时候一个星期回去一次，会买很多然后囤一下
  • Q:一个星期买一个星期的菜
  • A:对，但是绿叶菜什么的可能囤不了一个礼拜，所以还是去这边买，它那边因为每天都有食材的农药检测什么的，然后产品还是比较放心的
  • Q:也就说你工作之后才开始做饭
  • A:对，现在 买菜不都是app,京东也可以其实挺方便的，你也可以买这种原材料自己做，你也可以买半成品，现在还是蛮人性化的，半成品的话就是简单加热一下或者简单的加工一下就可以了
  • Q:我一开始工作的时候也是做饭后来老是把菜烧焦
  • A:那都有的呀，就像刚开始学烤东西学烘焙的人，他能把所有东西烤成黑色的，这个是很正常的，因为你要有这样一个过程，你是一个慢慢成长的一个过程特别是做饭，刚开始只是把它烧熟，再后来你可以发现可以把它烧得很好看，然后很好吃
  • Q:这是要多烧几次
  • A:当然这是检验兴趣的时候，有些人做过几次就没有兴趣了
  • Q:第一次做饭是什么时候
  • A:第一次的话在家里吧，做一点简单的比如下面条啊这些还是会的
  • Q:做的菜是你家乡的菜，还是自学的菜
  • A:不是，其实口味还是比较偏家里，但是现实时间比较长，因为现在很多APP啊，它每天都会有推荐，每天都会在上面看的，也会做上海这边的一些，像上海我比较喜欢吃的腌笃鲜，然后我本人比较喜欢吃辣的川菜这种口味
  • Q:那你觉得做饭难不难
  • A:不难，很多人会觉得那你下班了就已经很累了，为什么还要回去做饭，还要做每天中午的饭，但如果喜欢的话你会觉得没什么，如果你觉得真累的话就不会真做饭了
  • Q:你能说一下怎么新学一道菜呢
  • A:这时候一般，会在周末去尝试，因为时间比较充足，如果工序特别复杂的话我会一边开着APP做，其实现在做饭有一段时间了，所以也不需要开APP了，
  • Q:你这些菜你都记住了，
  • A:不是做那些大厨一样的菜基本上都可以，就看一下准备哪些材料，怎么做，或者自己改良，我现在豆果美食上有好几百个粉丝了，虽然我从来没有发过菜谱，之前的时候，便当啊什么的会在上面晒一下，就有粉丝关注你，其实这个感觉挺有成就感的，虽然我从来不会研究新的菜谱，也不像专门做饭的一些公众号什么的天天去发这些东西。
  • Q:我之前做饭的时候，发现网上菜谱材料买不到，你会有这方面的困惑吗
  • A:上海一般都能买得到，而且我在APP上，现在的APP其实对我有一定的引导性，因为确实是我没有见过的或是说我不经常吃的，但是它上面有时候经常会推荐之类的，然后就买过来尝一尝，所以在那个APP上买菜以后发现，菜做的种类其实是丰富的，如果我只是去边上的那个小的那个菜场，每天可能就固定的那些东西
  • Q:那你一般性做饭的话，做多大的量
  • A:一个人就做一个人的量，如果男朋友加班的话
  • Q:几菜几汤或者几荤几素这样
  • A:我一般吃素居多，第一是素食它比较快，因为它加工起来不像荤会时间比较长，然后因为工作的原因，接触的一些吃素的人，其实多多少少会受一点影响，因为觉得现在肉类受寄生虫啊生长环境之类的，会觉得吃素会更加有利于身体健康。我们有很多同事是素食主义者，不是因为减肥而是为了身体健康。研究表明吃素它可以多活几年减少生病啊 对抵抗力增强啊，并不是大家理解的那样如果不吃肉就营养不良，吃素食的人寿命会比正常人多七年好像
  • Q:因为我发现像中国的话，老年人吃素比较多，到老的时候会比较注意身体健康，吃斋念佛。。。
  • A:对其实和佛教还有一点点关系，因为我那个同事，他是个男生，身体也挺壮，他吃素是因为有一天在家带他女儿的时候，不小心把他女儿给磕到了缝了好几针，所以他当时为了他女儿吃素一年，赎罪。他吃素的过程中了解素食
    最后就没有佛教的思想了，最后他觉得吃素有利于身体健康，而且他吃素了一年，他感觉到了确实身体不一样了，但是我们还是没有和他一样吃素
  • Q:那你对食材有什么要求吗
  • A:要新鲜，能量越少越好，虽然说不可能没有能量，所以我会在那个APP上一直买菜，不会随便去外面买菜
  • Q:你学做饭的话是看视频还是图文
  • A:不看视频，视频浏览微博的时候有，因为会关注微博的账号，但是做饭的时候不回去看视频，视频里面全是简化的就是没有图片那么直观，大概一下就这么过去了，图文一般就是图片然后再那个文字说明
  • Q:每天都会想今天是做什么菜？
  • A:不用诶，还是那个APP帮我比较多吧，因为买菜的时候我就会在想这个礼拜我买这么多菜大概会怎么吃，然后比如鱼跟肉，今天吃鱼明天肯能会吃肉，我一般挺喜欢吃绿叶菜的，每天要有蔬菜，买菜的时候已经想的差别多了
  • Q:根据食谱来还是根据食材来制定一天的饭
  • A:没有想法的话就去看APP，因为它会说哪些菜现在特价呀，或者哪些菜是当季的呀，因为有时候确实不知道当季吃什么菜，但那个上会告诉你，一般会吃当季的多一点（时令蔬菜）
  • Q:比如我买了胡萝卜，你会搜胡萝卜做什么菜？
  • A:不用不用，现在已经完全不需要，大概只有最早最早的时候了，因为出去吃饭的时候就喜欢观察，你在外面吃，什么蔬菜和什么蔬菜搭配烧，这个我感觉已经不需要思考已经在大脑里了
  • Q:除了APP就是豆果上看菜谱
  • A:对，最早就是用豆果，后来他们给我推荐下厨房，我觉得也不需要了，因为你习惯了一个app以后你不会再去想用第二个同种功能的软件，除非特别特别差你才会换，所以豆果上我有粉丝了又习惯了
  • Q:豆果上哪些东西吸引你去用呢
  • A:图片拍的比较好，然后那个你不懂菜搜教程一般都有，豆果不知道什么时候开始的，它上面有商城你可以买东西
    ，我在上面也买过点，也没有便宜就感觉它里面有些东西比较好，比如同样式红豆，但它那个豆子是那种，新型创业就是产供销一体的没有中间商（直营）所以就觉得买他们的东西会健康放心，特别是我同事有做素食方面的活动然后就会接触一些有机食品，做社会创业的，还能解决家乡的问题然后有时候会经常买他们的东西，比如蜂蜜啊有什么坚果啊，有时候特别贵，像菌菇类的真的超级贵，但是好像还不错。
  • Q:我以前有同事江西的，他的同学在当地承包了一片橙子园，他在公司内部推销卖了加起来接近一百箱
  • A:买东西特别食行因为它除了在app上卖东西，它还有公众号每天推送普及一些知识，所以你关注它，一般它发的文章都会去看的，你天天吃的东西你都会知道是哪里产的，产地是什么样子的，有时候就会觉得很放心。

从上述记录可以得出：
Luo可以从烹饪以及相关利益人（同事、家人、朋友等）那里得到心理满足（成就感）。
同样互联网产品在她从事烹饪中起到至关重要的作用，帮助她在忙碌的工作时间中节约了时间和精力。
崇尚健康生活的Luo，对于食材有着严格的要求，新鲜绿色有机食品是她的首选，在食谱中有很大比例是绿色蔬菜，访谈中商家的折扣优惠和食品安全常常左右Luo在食材的挑选从而影响其日常饮食。
在社交属性在Luo从事烹饪中较为明显，其利益相关人从侧面影响巩固着她在烹饪上的持续性。
作为一个烹饪高手，在这个过程中同样减少了烹饪技能的学习成本。

总结

  • 1.菜谱插件

  • 2.用户概念模型
    Alisa是一名外企 白领，26岁，有个交往几年的男友，平时上班作息时间相对弹性，一般都会准时下班做饭。出于对食材新鲜和安全品质的追求，使用一些互联网产品来节约时间精力去做解决她的问题，差不多每隔一个礼拜的时间，她在app上会订购几天的食材，卖家会把这些东西存在离Alisa住所最近的冷藏储物柜里，她下班后，会前往那边取货，并且把它们存在自家的冰箱里，每天根据需要来安排当天的便当。
    受公司文化影响，她烹饪有很大的社交属性，每天中午都会和几个带饭的同事在公司大楼内的茶水间一起吃饭，一起讨论美食烹饪，一起聊聊八卦。热爱公益事业她经常参加公益事业活动，特别是支持一些绿色农产品的公益创业，Alisa重视健康饮食喜欢素食及有机食物，偏爱精致生活，喜欢把生活整理得井井有条。

  • 3.设计师和设计流程可以从模型中了解用户的职业，采购食物习惯，相关受益人，自己做饭的目的，做饭流程项目团队可以，设计团队可以从用户心理，竞品，及互联网技术对于用户做饭习惯的影响
